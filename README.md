- [x] Create account

- [ ] Confirm Secret (Login)
- [ ] Log in

- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow user
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload photos
- [ ] Edit the photo
- [ ] See the feed
